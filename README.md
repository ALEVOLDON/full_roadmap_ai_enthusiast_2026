## 🧭 HOW THIS ROADMAP WORKS

Это не теория. Ты двигаешься через создание реальных инструментов. 

---

## 🟢 Beginner Path
Если ты с нуля:

### Step 1 — База
- Что такое LLM
- Как работают API
- Prompt engineering

### Step 2 — Первый проект
👉 Сделай простой AI-инструмент (например, генератор текста или чат с моделью).

---

## 🟡 Developer Path
Если ты уже кодишь:

### Step 1 — Быстрый вход
- OpenAI / Gemini API
- LangChain / PydanticAI / LangGraph

### Step 2 — Проект
👉 **AI Telegram Bot** (с памятью, кастомными командами и интеграциями).

---

## 💰 Money Path
Если цель — доход:

### Step 1 — Быстрый продукт
👉 Сделай AI tool под конкретную рыночную задачу.

### Step 2 — Упаковка
- Лендинг
- Простой UI

### Step 3 — Монетизация
- Подписка или продажа как сервис (SaaS).

## 🔥 CORE STACK (2026)

### 🧠 AI

### ⚙️ Backend
- Node.js / Python
- Fast API / API Routes

### 🧩 Automation
- n8n / Zapier

### 🎨 Frontend
- React / Vite / No-code tools

## 🧪 REQUIRED PROJECTS
❗ Без них не двигаться дальше:
1. **AI Tool**: Мини-приложение с API.
2. **Telegram Bot**: С памятью и логикой.
3. **AI SaaS**: Простой сервис с UI и авторизацией.

## ❌ WHAT NOT TO LEARN
Сэкономь годы жизни:
- ❌ Глубокая математика (если не идешь в ресёрч)
- ❌ Классический ML в деталях
- ❌ Просмотр 100 курсов без практики

👉 Фокус: **build > learn**
